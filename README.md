# ocws-basic-pack
This package sets up the information for the OCWS Basic site. it offers the installation of all the plugins needed for the job; which are:

- OCWS Admin Bar
- OCWS Admin Bar Greeting
- GitHub Updater
- GitHub Link
- Login Logo
